Which one should I use?

Problem Best starting receipt Paid-pilot deliverable
An agent says a code review is done, but you cannot tell what was actually inspected. Audit receipt A reusable review workflow with pass/fail/incomplete criteria and a sample completed receipt.
A workflow fails only on some machines because a tool or profile is hidden. Profile exposure receipt A setup checklist plus receipt fields that expose required tools, visible tools, and blocked paths.
A large generated audit report looks impressive but may hide missing appendices or unverified claims. Large audit report receipt A report contract with verifier counts, appendix publication state, and drift-risk handling.
A team shares MCP or tool-call infrastructure and needs access/accountability records. MCP team audit receipt A team-specific receipt schema for identity, allowlist, redaction, and offboarding evidence.
A trace-backed analyzer says the user's implementation instruction caused ambiguity or rework. Instruction diagnosis receipt A diagnosis contract with evidence refs, quote state, negative control, extractor source, and supported/weak/incomplete verdict.
